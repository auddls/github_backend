Spring 이란? '객체지향원칙' 을 지키면서 개발할 수 있도록 돕는 JAVA 진영의 대표적인 백엔드 프레임워크이다. 스프링에서는 어플리케이션 전역에서 사용할 공용 객체인 스프링 빈을 사용하며 스프링 컨테이너(어플리케이션 컨텍스트트)라고 하는 공용 창고에 (객체)를 저장해두고,필요한 빈을 컨테이너에서 받아 사용하는 방식을 사용한다. 이때, 빈을 요구하는 객체 또한 스프링 빈이고 필요한 빈은 스프링 프레임워크가 자동으로 가져다준다. (빈이 아닌 객체가 빈을 요구하면 스프링 프레임워크가 자동으로 가져다주지 않는다). 스프링 빈을 스프링 컨테이너에 저장하는 방법은 설정파일을 작성하는 방법과 컴포넌트스캔을 사용하는 방법 2가지가 있다. 또한 의존성을 주입하는 방법은 생성자주입,필드주입,메서드주입 3가지가 존재한다.
